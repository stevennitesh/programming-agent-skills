# Skill Validation

Use this folder for per-skill validation plans and evidence.

Each skill validation note should answer:

- What runtime behavior changed?
- What task or transcript tests that behavior?
- What would count as improvement?
- What evidence was observed?
- What failed or remained uncertain?

Suggested path:

```text
docs/validation/skills/<skill-name>/<behavior-or-facet>.md
```

Do not store research here. Store evidence that a skill behaved better or worse.
