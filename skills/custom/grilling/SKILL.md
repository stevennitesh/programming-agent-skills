# Grilling

**Outcome:** a confirmed design tree, left unexecuted.

**Orient.** Walk the design tree dependency-first. Start with a compact plan recap and the root uncertainty. On later turns, recap the latest answer and reopen branches it invalidates.

**Find.** Answer factual questions from available sources and cite every load-bearing fact. The user owns every material decision; recommendations remain advisory.

**Pressure.** Challenge the unresolved assumption, dependency, failure mode, or tradeoff that unlocks the most downstream branches.

**Ask.** Put exactly one decision to the user. Recommend an answer and its decisive tradeoff, then wait.

Repeat **Pressure -> Ask** until **Confirm** or **Pause**.

**Confirm.** When every material branch is resolved or explicitly deferred, present the exit packet and wait. The exit is Confirmed only after the user confirms shared understanding and the next route is named.

**Pause.** When the next decision requires unavailable source facts or runnable evidence, name the exact question and its decision impact. Recommend `$research` for a cited note or `$prototype` for runnable evidence. Recommend `$handoff` when that work must cross into a fresh session and return here.

**Return.** Include confirmed decisions, explicit deferrals, source pointers, any evidence gap, and the next route. Return the packet to an invoking caller; otherwise report it and stop. Leave the plan unexecuted.
