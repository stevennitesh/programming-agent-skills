# Quick Override

[`SKILL.md`](SKILL.md) owns contracts, state roles, shared gates, and completion.

Use when the maintainer names the exact target state.

Read current roles and enough source context to produce the required comment or brief. Skip request verification and grilling.

The command approves the named outcome and stated mutation packet. Keep the normal ready brief, disclaimer, `.out-of-scope/`, close, and read-back gates.

For `implemented`, require an explicit label-correction request and existing closeout evidence.
