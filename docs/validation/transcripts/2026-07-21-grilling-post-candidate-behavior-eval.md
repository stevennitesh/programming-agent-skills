  checkout hash above.

  context only when package bytes, protocol, inputs, tools, authority, and
  runtime were unchanged.
- Metadata decisions were locked before the worker read the body.
- Workers received fixed cases and returned compact action/Return packets; they
  did not receive the rubric, expected outcomes, candidate conclusions, or peer
  outputs.

  were reused unchanged and only affected candidate cases C and E reran.
- All workers used `fork_turns="none"`, inherited the available model and
  reasoning configuration, had read-only task authority, and were instructed
  to read only the assigned package. Exact model identifier, reasoning tier,
  token counts, and elapsed time per sample were unavailable.
- Root inspected every returned packet and alone applied the rubric.
