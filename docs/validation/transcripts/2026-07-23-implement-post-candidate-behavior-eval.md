## Fixed Protocol

Five fresh-context, read-only direct-agent samples ran per arm with inherited
root model, reasoning settings, runtime, and tools; no model override was used.
Each sample read only its assigned complete package, the shared engineering
contract, and the Finding Contract when needed. Control workers could not read
the candidate, synthesis, tests, evaluations, or peer outputs; candidate
workers could not read the control/canonical package, synthesis, tests,
evaluations, or peer outputs. No context was reused across arms.

Both arms received the same nine-case packet and output schema. Workers made no
mutations and invoked no downstream skill. Root retained the rubric and judged
every result. Raw fresh-context outputs remain attached to these direct task
records:

- control: `implement_control_1` through `implement_control_5`;
- candidate: `implement_candidate_1` through `implement_candidate_5`.

Exact backend build, token, and latency telemetry was unavailable.
