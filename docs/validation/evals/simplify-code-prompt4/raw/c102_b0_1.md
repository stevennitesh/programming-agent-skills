# C1-02 Characterization — Simplify Code read-only

## Positive

**Decision:** Cut.

The authoritative, versioned protocol contract fixes caller-visible input and
output without creating a new commitment. Before mutation, add the smallest
caller-facing characterization derived from that contract, observe it passing,
and confirm that it fails when the proposed branch deletion is applied alone.
That proof licenses the bounded deletion once the same characterization passes
afterward.

**Proof boundary:** the protocol-covered caller input/output exercised through
the existing caller-facing seam. It does not establish behavior outside the
contract, unexercised parser cases, or unrelated compatibility commitments.

**State:** read-only decision; no production mutation, staging, or external
state change. Repository work state was outside the packet and was not
inspected.

**Return:** bounded branch deletion is licensed, contingent on the stated
before-mutation, mutation-detection, and after-mutation proof sequence. The
preserved contract is the versioned protocol behavior; the residual proof owner
is the existing caller-facing seam.

## Negative

**Decision:** **No safe simplification**.

The proposed assertion mirrors a private helper's branch layout, cannot detect
a change in caller-visible output, and requires a new public hook. It therefore
provides no caller-facing behavior lock and crosses into a new proof-seam and
public-interface decision.

**Proof boundary:** none credible for the proposed cut. A safe boundary would
need an existing caller-facing observable tied to an authoritative behavior
commitment; the private-layout assertion supplies neither.

**State:** read-only decision; no production mutation, new hook, staging, or
external state change. Repository work state was outside the packet and was not
inspected.

**Return:** no cut is licensed. The exact proof gap is the absence of a
caller-visible assertion capable of detecting changed output. Return the new
interface or proof-seam decision to Codebase Design; otherwise retain the
branch.
