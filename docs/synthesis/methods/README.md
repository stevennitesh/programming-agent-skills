# Synthesis Methods

Methods are reusable ways to turn research into compact skill behavior.

Use this folder for process docs that apply across multiple skills or skill
families.

## Files

| File | Role |
| --- | --- |
| [`source-to-skill-flow.md`](source-to-skill-flow.md) | Production flow from intent and source search through detailed draft, plain-language candidate, reality validation, and final prune. |
| [`prompts/`](prompts/) | Reusable prompts for each source-to-skill step. |
| [`plain-language-pass.md`](plain-language-pass.md) | Post-draft plain-language pass for blunt gates and clear completion criteria. |

## Boundary

- Put reusable synthesis methods here.
- Put per-skill design judgment in `../skills/`.
- Put family-level design judgment in `../families/`.
- Put source research in `../../research/`.
- Put behavior evidence in `../../validation/`.
