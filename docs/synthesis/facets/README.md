# Facet Synthesis

This folder holds skill/facet synthesis timelines.

Use one folder per skill and one folder per facet:

```text
docs/synthesis/facets/<skill-name>/
  README.md
  FACET-<n>-<NAME>/
    06-agent-bridge.md
    07-full-behavior-synthesis.md
    08-candidate-runtime-draft.md
    09-detailed-skill-context-draft.md
    10-plain-language-validation-candidate.md
    11-validation.md
    12-final-prune.md
```

Research stays in `docs/research/`. Runtime behavior stays in
`skills/custom/`. This folder owns the synthesis and integration artifacts
between those boundaries.
