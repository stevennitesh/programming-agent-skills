# Research

Research docs are source intake and candidate-language extraction. They can be
verbose because they are not the final skill text.

Use this folder to answer:

- What sources deserve to shape the skill pack?
- What professional or agentic behavior should a source recruit?
- What terms, rules, failure modes, and evidence gates should move into
  synthesis?

## Lanes

| Lane | Role |
| --- | --- |
| [`sources/`](sources/) | Ranked books, papers, manuals, official docs, and source-family rationale. |
| [`language/`](language/) | Exploratory language inventory, upstream audits, high-signal words, and bridge vocabulary. |
| [`skill-facets/`](skill-facets/) | Per-skill research maps, reusable facet-research prompt, search vocabulary, and facet packets. |
| [`backlog/`](backlog/) | Deferred research opportunities, stop rules, and gap-pass notes. |

## Rule

Research should feed synthesis. If a source does not sharpen a leading word,
rule, failure mode, evidence gate, stop/ask condition, or review check, keep it
out of the active skill path.
